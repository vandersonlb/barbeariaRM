## Objetivo
Criação de um site estático com o intuito de colocar em prática os conceitos e tecnologias aprendidos até então: 
- Design Web
- Usabilidade
- Acessibilidade
- Performance
- HTML, CSS e Javascript
- Responsividade
- Sass
- Webpack

### Dia 1
- Criação e prototipagem do layout
> Ferramentas: Adobe XD, Photoshop e Illustrator

### Dia 2
- Iniciando a base do site
- Reset CSS
- Mixins e variáveis do Sass
- Bloco: Header (com menu hambúrguer no mobile)
> Ferramentas: HTML, CSS e Sass

### Dia 3
- Blocos: Banner, Estilos e Sobre mim
- Fix do menu hambúrguer (não desaparecia quando clicado)
> Ferramentas: HTML, CSS, Sass e Javascript

### Dia 4
- Blocos: Valores, Fotos, Contato e Footer
> Ferramentas: HTML, CSS e Sass

### Dia 5
- Sessão fotos sendo alimentada pelo Instagram
- Skeleton nos blocos das fotos enquanto as fotos não carregam
- Botão "Ver mais fotos" carrega mais 4 fotos e depois disso direciona para o Instagram
> Ferramentas: HTML, CSS, Sass, Javascript e [InstagramFeed](https://github.com/jsanahuja/InstagramFeed)

### Dia 6
- Galeria de fotos tela cheia estilo Carrossel Infinito
> Ferramentas: HTML, CSS, Sass e Javascript
